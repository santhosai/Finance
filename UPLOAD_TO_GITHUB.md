# Upload Files to GitHub (Web Method)

## Step 1: Go to Your Repository
https://github.com/santhosai/Finance

## Step 2: Upload These Files (Drag & Drop)
1. Click "uploading an existing file"
2. Drag these files from your folder:
   - `app_production.py`
   - `requirements.txt` 
   - `Procfile`
   - `railway.json`
   - `runtime.txt`
   - `nov 11.xlsx`
   - `templates/index.html`
   - `static/bg.jpg`

## Step 3: Commit Files
- Add commit message: "Initial upload"
- Click "Commit changes"

## Step 4: Deploy to Railway
1. Go to https://railway.app
2. Sign up with GitHub
3. Click "New Project" 
4. Select "Deploy from GitHub repo"
5. Choose "santhosai/Finance"
6. Railway auto-deploys!
7. Get your public URL
